# 기능설명
- Paing : 선그리기<br>
- 선굵기 조절하기<br>
- Fill : 캔버스 전체 색채우기<br>
- Save : 작품 저장<br>
- Erase All : 캔버스 초기화<br>

# 후기
- 자바스크립트의 무궁무진함을 새삼 느꼈습니다.<br>그려진 도형 안에 색 채우기, 3d 그리기 등도 구현해보고 싶다는 생각이 들었습니다.
